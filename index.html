マイイレ - サッカーの選手評価・記録アプリ

はじめに

本アプリは、クラブ（クラブ全体を管理するクラブサポーター向け） と 選手個人（1人の選手に集中して管理する個人サポーター向け）の2つの軸で評価を記録・可視化できます。

このページは、インストール直後に迷わないための実用的な操作ガイドです。まずは、ご自身の用途に近い方から始めてみてください。

クラブで管理したい方 クラブメニューの使い方へ クラブ登録 → 選手登録 → 試合登録 → 評価 → 画像生成 個人選手を記録したい方 選手個人メニューの使い方へ 選手登録 → 試合登録 → 評価 → プロフィール/履歴 → 画像生成

宣伝を兼ねた全体紹介は note記事 をご覧ください。

クラブメニュー【クラブサポーター向けの流れ】

クラブ単位で選手を管理し、試合ごとに評価。選手一覧や試合ごとの評価を画像生成してSNSなどで共有できます。

  1. Step 1. ダウンロード

    App Store からアプリをインストール。

  2. Step 2. クラブ登録

    ホーム → クラブ追加（「＋ 新規」ボタン）

    背景色・文字色をクラブカラーに設定すると、画像生成の配色に反映されます。

    色は後から変更可能です。文字色は可読性重視で調整をしてください。

    続いて、シーズンを追加します。（「＋ 新規」ボタン）

    シーズン名を入力して、保存します。

  3. Step 3. 選手登録

    ホーム → クラブ名 → 選手設定（「⚙」ボタン）

    続いて、選手を追加します。（「＋ 選手追加」ボタン）

    シーズン単位で管理できます。
    過去シーズンからの引き継ぎも可能で、更新が簡単です。

  4. Step 4. 試合登録

    ホーム → クラブ名 → 評価入力

    評価入力画面に遷移します。

    試合を追加（「＋」ボタン）します。

    試合名を入力してください。

    試合名は後から変更可能です。日付や試合名の入力でわかりやすくなります。

  5. Step 5. 評価入力（★）

    ホーム → クラブ名 → 評価入力

    試合名が入力されていることを確認して各選手の「未入力」欄をタップして星評価を入力できます。

    選手名をタップすると、各選手に対して試合の一言メモ(50文字まで)を残せます。

    短文でもOK。後で振り返りやすくなります。

    「👑」ボタンからMOM（マン・オブ・ザ・マッチ）も選出できます。

  6. その他機能 選手詳細（統計）

    ホーム → クラブ名 → 選手一覧 → 選手詳細（各選手をタップ）

    平均評価やMOM回数を自動集計します。
    折れ線グラフで評価推移（直近5試合／全試合）を切替表示。

    その他機能 画像生成（共有）

    ホーム → クラブ名 → 投稿用（タブバー「カメラ」ボタンをタップ）

    試合ごとの評価まとめ、選手一覧、選手詳細などを画像として保存。
    クラブカラーを活かしたデザインでSNSシェアに最適です。

    縦長画像はInstagramで2〜3分割して投稿すると見やすく表示できます。

補足：バックアップ／復元

  • 端末変更時は、今後のアップデートでエクスポート機能を提供予定です。
  • 同一端末内のデータはアプリを削除すると消えますのでご注意ください。

選手個人メニュー【選手個人のサポーター向けの流れ】

個人の選手だけに焦点を当て、試合ごとにより細かく評価。プロフィールや試合ごとの評価を画像生成してSNSなどで共有できます。

  1. Step 1. ダウンロード

    App Store からアプリをインストール。

  2. Step 2. 選手の登録

    ホーム → 選手を追加（「＋ 新規」ボタン）

    名前・背番号・ポジションを設定します。

    背景色・文字色をプレイヤーカラーに設定すると、画像生成の配色に反映されます。

    色は後から変更可能です。文字色は可読性重視で調整をしてください。

    続いて、シーズンを追加します。（「＋ 新規」ボタン）

    シーズン名を入力して、保存します。

  3. Step 3. 試合登録

    ホーム → 選手名 → 試合記録入力（タブバー「記録」ボタン）

    試合記録入力画面に遷移します。

    試合を追加（「＋」ボタン）をします。

    試合名を入力してください。

    試合名は後から変更可能です。日付や試合名の入力でわかりやすくなります。

  4. Step 4. 評価入力（★）

    ホーム → 選手名 → 試合記録入力（タブバー「記録」ボタンをタップ）

    試合名が入力されていることを確認して「未入力」欄をタップして星評価を入力できます。

    試合の一言メモ(20文字まで)を残せます。

    一言メモは履歴一覧で振り返ることができます。

    タグを付けて管理すると後から検索しやすくなります。

    印象に残ったことや結果などをタグで管理します。

    さらに試合の詳細メモ(200文字まで)も残すことができます。

  5. その他機能 プロフィール入力

    ホーム → 選手名 → 選手プロフィール画面（タブバー「選手」ボタンをタップ）

    プレイヤーノートに入力して選手の特徴や魅力、目標などを整理。

    平均評価や記録試合数を自動集計します。
    折れ線グラフで評価推移（直近5試合／全試合）を切替表示。

    その他機能 履歴確認

    ホーム → 選手名 → 選手プロフィール画面（タブバー「選手」ボタンをタップ）

    評価履歴を一覧で確認できます。タグで絞り込みも可能。

    その他機能 画像生成（共有）

    ホーム → 選手名 → 投稿用（タブバー「カメラ」ボタンをタップ）

    試合ごとの選手プロフィールや試合の評価を画像として保存。
    プレイヤーカラーを活かしたデザインでSNSシェアに最適です。
