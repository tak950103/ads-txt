マイイレ - サッカーの選手評価・記録アプリ

はじめに

「アプリの紹介」は note記事 に掲載しています。ここでは、実際に使い始めるための詳細な手順を説明します。

  • 画像はイメージです。バージョンにより表示が異なる場合があります。

ガイド一覧

  1. クラブ機能（クラブサポーター向け）：ダウンロード → クラブ登録 → 選手登録 → 試合登録 → 評価　その他：選手詳細、画像生成
  2. 選手個人機能（個人サポーター向け）：ダウンロード → 選手登録 → 試合登録 → 評価 その他：プロフィール入力、履歴確認、画像生成

クラブサポーター向けの流れ

クラブ単位で選手を管理し、試合ごとに採点。クラブカラーで画像を生成してSNSで共有できます。

  1. Step 1. ダウンロード

    App Store からアプリをインストール。

  2. Step 2. クラブ登録

    ホーム → クラブ追加（「＋ 新規」ボタン）

    背景色・文字色をクラブカラーに設定すると、画像生成の配色に反映されます。

    色は後から変更可能です。文字色は可読性重視で調整をしてください。

    続いて、シーズンを追加します。（「＋ 新規」ボタン）

    シーズン名を入力して、保存します。

  3. Step 3. 選手登録

    ホーム → クラブ名 → 選手設定（「⚙」ボタン）

    続いて、選手を追加します。（「＋ 選手追加」ボタン）

    シーズン単位で管理できます。
    過去シーズンからの引き継ぎも可能で、更新が簡単です。

  4. Step 4. 試合登録

    ホーム → クラブ名 → 評価入力 → 試合追加（「＋」ボタン）。

    試合名を入力してください。

    試合名は後から変更可能です。

  5. Step 5. 評価入力（★）

    ホーム → クラブ名 → 評価入力

    試合名が入力されていることを確認して各選手の「未入力」欄をタップして星評価を入力できます。

    選手名をタップすると、各選手に対して試合の一言メモ(50文字まで)を残せます。

    短文でもOK。後で振り返りやすくなります。

    「👑」ボタンからMOM（マン・オブ・ザ・マッチ）も選出できます。

  6. その他機能 選手詳細（統計）

    ホーム → クラブ名 → 選手一覧 → 選手詳細（各選手をタップ）

    平均評価やMOM回数を自動集計します。
    折れ線グラフで評価推移（直近5試合／全試合）を切替表示。

    その他機能 画像生成（共有）

    試合ごとの評価まとめ、選手一覧、選手詳細などを画像として保存。
    クラブカラーを活かしたデザインでSNSシェアに最適です。

    縦長画像はInstagramで2〜3分割して投稿すると見やすく表示できます。

補足：バックアップ／復元

  • 端末変更時は、今後のアップデートでエクスポート機能を提供予定です。
  • 同一端末内のデータはアプリを削除すると消えますのでご注意ください。

② 個人選手向けの流れ

自分のプレー記録を残したい方向け。試合ごとに自己評価し、プロフィールや履歴をまとめて画像化できます。

  1. Step 1. ダウンロード

    App Store からインストール。

  2. Step 2. 選手個人の登録

    ホーム → 個人選手を追加。名前・背番号・ポジションを設定。

    プレイヤーカラーを設定するとプロフィール画像が映えます。

  3. Step 3. 試合登録

    試合を追加し、日付・試合名などを記録。

  4. Step 4. 評価入力（★）・メモ・タグ

    試合を選択し、星評価を入力。短いメモやタグを付けて管理すると後から検索しやすくなります。

  5. Step 5. プロフィール入力

    選手プロフィールにノートを追加して自分の特徴・課題・目標を整理。

  6. Step 6. 履歴確認（直近／全期間）

    評価履歴を一覧で確認。シーズンやタグで絞り込みも可能。

  7. Step 7. 画像生成（共有）

    プロフィール画像、試合記録画像、コメント付き画像などを生成して保存・共有。

    「直近5試合／全試合」を選んで推移グラフを含めた画像出力も可能です。
